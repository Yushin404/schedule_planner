共有の考え方

この版は data/shared-schedule.json を全員で読む作りです。 GitHub上のJSONを更新すると、URLにアクセスした全員に同じ内容が表示されます。

ただし、GitHub Pages単体ではブラウザから直接GitHubへ安全に書き込めません。 そのためこの版は 共有閲覧 + 管理者がJSONを更新 方式です。

管理者用

ダウンロードしたJSONを GitHub の data/shared-schedule.json に上書きすると共有内容を更新できます。
