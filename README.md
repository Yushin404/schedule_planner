# Schedule Planner

送迎スケジュール管理用のシンプルな静的Webアプリです。

## フォルダ構成

```text
schedule-planner/
├─ index.html
├─ css/
│  └─ style.css
├─ js/
│  ├─ main.js
│  ├─ config.js
│  ├─ data/
│  │  ├─ storage.js
│  │  └─ schedule.js
│  ├─ ui/
│  │  ├─ stats.js
│  │  └─ table.js
│  └─ utils/
│     ├─ date.js
│     └─ dom.js
└─ README.md
```

## 使い方

1. `index.html` をブラウザで開く
2. 予定・実績・送迎時刻・補足を入力する
3. データはブラウザ内に自動保存される

## GitHub Pages に置く場合

このままリポジトリにアップして、`index.html` をルートに置けばそのまま公開できます。
